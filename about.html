About This Project

The Massachusetts Mosques Directory is a student-built project created for the University of Massachusetts Boston. The purpose of this website is to provide clear, accessible information about masajid (mosques) across the state, while also offering useful Islamic learning tools for Muslims and non-Muslims alike.
